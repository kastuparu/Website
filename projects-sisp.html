Senior Independent Study Project (SISP)

Digestive System Tracking App

Mass Academy seniors complete a 100-hour independent study project on a topic of their choice. I chose to use my independent study project to continue building my software development skills. My final presentation for this project is on the right and my code is on GitHub.

My project was creating a mobile iOS app for tracking digestive symptoms. Throughout this project, I learned how to use Swift and SwiftUI, which are tools for developing iOS apps. I also learned how to use and integrate a SQLite database for an application. I also learned more about time and project management.
